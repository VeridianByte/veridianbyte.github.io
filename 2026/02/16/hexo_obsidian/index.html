obsidian配置

  1. 在 https://obsidian.md/download 下载并安装obsidian
    安装完成后打开obsidian选择my-blog作为本地仓库打开：
  2. 安装插件
    在设置的第三方插件处，关闭安全模式

    在社区插件市场安装 File Tree Alternative、floating toc、Git、Shell commands、Custom Attachment Location。
  • 设置Shell commands，选择New shell command，添加start deploy.sh和start server.sh

    在my-blog目录中添加deploy.sh与server.sh

server.sh：

在命令面板中搜索start deploy.sh或start server.sh即可同步GitHub或本地预览

  • 设置Custom Attachment Location，新附件位置填入./${noteFileName}
  1. 将博客根目录_config.yml文件的post_asset_folder值设置为true
  2. 打开设置在文件与连接选项中按图中设置。附件文件夹位置不用填，由Custom Attachment Location插件自动生成。
